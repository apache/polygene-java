TransientComposite tutorials

Throughout this set of tutorials it will be shown how to create and work with TransientComposites, which is the basic element in Apache Zest™. We will refactor one HelloWorld class to take advantage of the various features in Zest™. These refactorings will make it easier to reuse parts of the class, and introduce new features without having to change existing code. We will also look at some of the existing classes, or Fragments, available in Zest™ that you can reuse so that you don't have to write everything yourself.

Each tutorial in this series starts with the result from the previous tutorial, so you can always look at the next tutorial step for guidance on what to do. Here are the steps:

  1. Interface refactoring
  2. Creating a TransientComposite
  3. Mixins
  4. Concerns
  5. Constraints
  6. SideEffects
  7. Properties
  8. Generic mixins
  9. Private and abstract mixins
