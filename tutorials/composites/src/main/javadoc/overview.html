Apache Polygene™ tutorials

Composite tutorials

Throughout this set of tutorials it will be shown how to create and work with Composites, which is the basic element in Polygene™. We will refactor one HelloWorld class to take advantage of the various features in Polygene™. These refactorings will make it easier to reuse parts of the class, and introduce new features without having to change existing code. We will also look at some of the existing classes, or Fragments, available in Polygene™ that you can reuse so that you don't have to write everything yourself.

Each tutorial in this series starts with the result from the previous tutorial, so you can always look at the next tutorial step for guidance on what to do.
